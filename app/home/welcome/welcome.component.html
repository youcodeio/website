We are like you, developers. We love learning new frameworks by ourselves. Youtube offers hundred of talks and tuts, but it could be difficult to find. This is what YouCode is all about.

Powered by
Google

We are using all the latest technologies provided by Google to bring YouCode online. We're using Google App Engine to host the app written in Golang, Youtube API to search through it and AngularJS for easy rendering.

Open-source
Project

We strongly believe that open-source code help everyone, that's why the whole project is hosted on Github. Even the server-side code. You can fork it, and do whatever you want, especially learn !
